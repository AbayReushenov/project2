If two values are given
they are separated by "/".
The grid-column-start longhand
is set to the value before the slash,
and the grid-column-end longhand is set
to the value after the slash.
Two

а максимальная в "auto".
Grid-элементы располагаются в сетке линейно.
.two { grid-column: 2 / 4;
grid-row: 1 / 3; }
